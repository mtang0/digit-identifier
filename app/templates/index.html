Digit Identifier

An AI Program Designed to Identify Digits in Pictures

Start Classifying

Project Overview

Our model takes in photos of individual handwritten digits and accurately identifies them with a bounding box.

How to Get the Best Results

Our model performs best under certain circumstances:

  • Medium sized square image
  • Black backgroud with a white digit
  • .JPG, .JPEG, .PNG file
  • One or two digits in the picture

Applications

Our model can be used to automate the conversion of handwritten data to a computer program, such as:

  • Reading serial numbers
  • Reading license plates
  • Auto-grading
  • Form data entry
  • Bank check processing

Resources

MNIST

MNIST is a large pre-built dataset that consists of thousands of handwritten digits for 0 - 9, we used this dataset to train our model and without it image acquisition would have been much more difficult.  

Labelbox

Labelbox is a tool we used to help label our dataset into the 10 digit categories from 0-9. Without Labelbox we wouldn’t h ave been able to tell the computer what digit the image showed, or build the training set. 

YOLO

YOLO is an object detection repository which uses computer vision algorithms to identify objects in an image. We used YOLO to train our model to find and recognize digits. 

AI Camp
